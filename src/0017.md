### create-react-app

快速创建一个最简单的react项目

环境：
- node 
- cnpm：https://npmmirror.com/  或者 npm

安装命令：
- npm install -g create-react-app

创建React项目：
- create-react-app demo

### react-scripts

正是因为有了react-scripts的集成化能力，项目目录才会如此简洁。
但这也限制了灵活性，使得我们很难对其内部做出修改。
而eject可以将react-scripts中几乎全部的逻辑代码，释放到项目的根目录中，从而允许我们自行修改。
